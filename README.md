ClassUp — Calendario Escolar Social

ClassUp es una aplicación web creada para estudiantes de nivel secundario.  
Combina un calendario escolar, recordatorios automáticos y un muro social donde los alumnos pueden publicar, comentar y dar like a actividades del colegio.  
Su propósito es mejorar la organización académica, evitar olvidos y fortalecer la comunicación entre compañeros.

Problema
Los estudiantes suelen olvidar tareas, fechas de exámenes y actividades importantes.  
No existe un espacio centralizado donde esté *toda* la información escolar, y los métodos tradicionales (cuadernos, notas del celular) son fáciles de perder u olvidar.

---

Solución: ClassUp
ClassUp ofrece:

- Calendario escolar interactivo
- Publicaciones y recordatorios
- Comentarios con nombre automático del usuario
- Notificaciones personalizadas
- Sistema de amigos con recordatorios compartidos
- Perfil editable  

Todo en un entorno visual moderno, simple y pensado para estudiantes.

Cómo Usar el Sistema

1. Registrarse: creando una cuenta.  
2. Iniciar sesión para acceder a todas las funciones.  
3. Publicar recordatorios desde la página principal  
   *Los posts aparecen automáticamente en inicio.php con su diseño incorporado.*  
4. Comentar en publicaciones de otros usuarios.  
5. Agregar amigos y activar recordatorios compartidos.  
6. Usar el calendario para ver fechas importantes.  
7. Editar el perfil con foto y datos personales.
